# ✈ Airline Passenger Satisfaction Prediction

## 🚀 Live App

👉 https://huggingface.co/spaces/jas1609/airline-passenger-satisfaction

---

## 📌 About the Project

This project focuses on predicting whether an airline passenger is satisfied or not based on various service and travel-related factors.

The goal was to build a complete end-to-end machine learning system — starting from data analysis to deploying a live application that anyone can use.

---

## 🧠 What I Built

* Performed detailed **EDA** to understand passenger behavior
* Built multiple machine learning models and compared performance
* Used **Random Forest** as the final model
* Implemented **manual preprocessing** (encoding, scaling, feature alignment)
* Designed a clean and interactive **Streamlit UI**
* Deployed the app on Hugging Face for real-time usage

---

## ⚙️ Tech Stack

* Python
* Pandas & NumPy
* Scikit-learn
* Streamlit
* Hugging Face Spaces

---

## 📊 Model Details

* Model Used: Random Forest Classifier
* Features: Passenger details + service ratings
* Output: Satisfaction prediction with confidence score
